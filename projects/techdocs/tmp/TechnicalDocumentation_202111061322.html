The ln Command

The ln command is used to create links between files. See the examples of its usage below:

  • ln -s
    Creates a symbolic(soft) link(pointer) to a file. If the file is deleted the link stops working. 
    
┌──(kali㉿aBANANA)-[~]
└─$ ln -s Documents/myfile.txt .

┌──(kali㉿aBANANA)-[~]
└─$ ls -l myfile.txt
lrwxrwxrwx 1 kali kali 20 Nov 6 06:51 myfile.txt -> Documents/myfile.txt

┌──(kali㉿aBANANA)-[~]
└─$ cat myfile.txt
aBANANA

┌──(kali㉿aBANANA)-[~]
└─$ rm Documents/myfile.txt

┌──(kali㉿aBANANA)-[~]
└─$ cat myfile.txt
cat: myfile.txt: No such file or directory

┌──(kali㉿aBANANA)-[~]
└─$
  • ln -P file1.txt filetwo.txt
    Make a hard link called 'filetwo.txt' to the underlying INODE entry for the file 'file1.txt'. If 'file1.txt' is deleted, 'file2.txt' will still contiue working. 
    
┌──(kali㉿aBANANA)-[/]
└─$ touch tmp/myfile.txt

┌──(kali㉿aBANANA)-[/]
└─$ echo 'aBANANA' > tmp/myfile.txt

┌──(kali㉿aBANANA)-[/]
└─$ sudo ln -P tmp/myfile.txt . 
[sudo] password for kali:

┌──(kali㉿aBANANA)-[/]
└─$ ls -l myfile.txt
-rw-r--r-- 2 kali kali 8 Nov 6 07:05 myfile.txt

┌──(kali㉿aBANANA)-[/]
└─$ cat myfile.txt
aBANANA

┌──(kali㉿aBANANA)-[/]
└─$ rm tmp/myfile.txt

┌──(kali㉿aBANANA)-[/]
└─$ cat myfile.txt
aBANANA

┌──(kali㉿aBANANA)-[/]
└─$
